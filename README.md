# The Guardian - Crypto Price Tracker (Nexus)

Професійний скрипт на Python для відстеження ціни криптовалюти у реальному часі через **Bitget API**. Проект має модульну структуру, високу стійкість до збоїв та вбудований технічний аналіз.

## Функції
- 📈 **Відстеження ціни**: Отримання актуальної ціни кожні 30 секунд через Bitget API.
- 📊 **Технічний аналіз**: Моніторинг індикатора **RSI(14)** на таймфреймі **M5** (5 хвилин).
- 🎨 **Кольорова індикація**: 
  - **Зелений**: зміна ≥ +0.1%
  - **Червоний**: зміна ≤ -0.1%
  - **Білий**: незначні коливання (< 0.1%)
- 🌀 **Анімоване очікування**: Динамічний спінер `[| / - \]` під час паузи між запитами.
- 🛡️ **Стійкість до помилок**: Якщо біржа Bitget не відповідає, скрипт чекає 5 секунд і пробує знову (Retry logic).
- 📝 **Логування**: Всі події, ціни та RSI записуються у файл `nexus.log`.
- 🧩 **Модульність**: Код розділений на `main.py`, `utils.py` та `logger.py`.

## Структура проекту
- `main.py` — основний цикл програми та робота з API.
- `utils.py` — математичні розрахунки (RSI) та анімація.
- `logger.py` — налаштування системи логування.
- `nexus.log` — файл з історією цін та технічних показників.

## Вимоги
- Python 3.x
- Бібліотека `requests`

## Встановлення
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/NORDY-techno/Nexus.git
   cd Nexus
   ```
2. Встановіть залежності:
   ```bash
   pip install requests
   ```

## Запуск
```bash
python main.py
```
