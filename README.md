# The Guardian - Crypto Price Tracker

Простий скрипт на Python для відстеження ціни криптовалюти (ETH/USDT) у реальному часі через публічне API Binance.

## Функції
- 📈 Отримання ціни кожні 30 секунд.
- 📊 Розрахунок зміни ціни у відсотках порівняно з попереднім запитом.
- 🎨 Кольорова індикація:
  - **Зелений**: ціна зросла.
  - **Червоний**: ціна впала.
  - **Білий**: без змін.
- 🔇 Тихий перший запуск (починає вивід з другого запиту).

## Вимоги
Для роботи скрипта потрібен Python 3 та бібліотека `requests`.

## Встановлення
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/NORDY-techno/Nexus.git
   cd Nexus
   ```
2. Встановіть необхідну бібліотеку:
   ```bash
   pip install requests
   ```

## Запуск
Запустіть основний файл:
```bash
python main.py
```

## Як це працює (для початківців)
- **Цикли**: Використовується `while True` для постійної роботи програми.
- **Змінні**: Зберігають поточну та попередню ціну для порівняння.
- **API**: Скрипт звертається до серверів Binance, щоб отримати найсвіжіші дані без реєстрації та ключів.
