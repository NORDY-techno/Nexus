# The Guardian - Crypto Price Tracker (Nexus)

Професійний скрипт на Python для відстеження ціни криптовалюти (ETH/USDT) у реальному часі через публічне API Binance. Проект має модульну структуру та високу стійкість до збоїв.

## Функції
- 📈 **Відстеження ціни**: Отримання актуальної ціни кожні 30 секунд.
- 📊 **Аналіз змін**: Розрахунок зміни ціни у відсотках порівняно з попереднім запитом.
- 🎨 **Кольорова індикація**: 
  - **Зелений**: зміна ≥ +0.1%
  - **Червоний**: зміна ≤ -0.1%
  - **Білий**: незначні коливання (< 0.1%)
- 🌀 **Анімоване очікування**: Динамічний спінер `[| / - \]` під час паузи між запитами.
- 🛡️ **Стійкість до помилок**: Якщо біржа не відповідає, скрипт чекає 5 секунд і пробує знову (Retry logic).
- 📝 **Логування**: Всі події та помилки записуються у файл `nexus.log` з мітками часу.
- 🧩 **Модульність**: Код розділений на логічні частини (`main.py`, `utils.py`, `logger.py`).

## Структура проекту
- `main.py` — основний цикл програми.
- `utils.py` — допоміжні функції (анімація очікування).
- `logger.py` — налаштування системи логування.
- `nexus.log` — файл з історією подій.

## Вимоги
- Python 3.x
- Бібліотека `requests`

## Встановлення
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/NORDY-techno/Nexus.git
   cd Nexus
   ```
2. Встановіть залежності:
   ```bash
   pip install requests
   ```

## Запуск
```bash
python main.py
```
